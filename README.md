# ONG Sementes do Futuro

Projeto desenvolvido como atividade prática de estilização e leiautes, aplicando HTML, CSS3 e JavaScript para criar uma interface profissional, responsiva e acessível para uma ONG fictícia que apoia crianças em situação de vulnerabilidade.

## Funcionalidades

- Menu de navegação responsivo com submenu e versão mobile (hambúrguer)  
- Seção de destaque com imagem hero e chamada para ação  
- Cards responsivos para exibir projetos da ONG  
- Formulário de cadastro de voluntários com validação visual  
- Botões com estados visuais (hover, focus, active, disabled)  
- Sistema de cores, tipografia e espaçamento modular usando CSS custom properties  

## Tecnologias utilizadas

- HTML5  
- CSS3 (Flexbox, Grid, variáveis CSS)  
- JavaScript (validação de formulário e feedback visual)  

## Como usar

1. Clone ou baixe o repositório:  
```bash
git clone https://github.com/Mimis20/ong-plataforma-entrega-4.git
```  
2. Abra o arquivo `index.html` no navegador de sua preferência  
3. Navegue pelo menu e explore as páginas de projetos e cadastro
